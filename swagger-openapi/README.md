# Swagger API in Golang

```shell
go install github.com/go-swagger/go-swagger/cmd/swagger@latest
# GO111MODULE=off go get -u github.com/go-swagger/go-swagger/cmd/swagger
swagger serve -F=swagger swagger.yaml
```
